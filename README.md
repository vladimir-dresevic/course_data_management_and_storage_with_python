# Data Management & Storage with Python

This repository contains **lecture examples**, **hands-on exercises**, and **mini-projects** for the course **Data Management & Storage with Python**.  
You’ll learn practical ways to **read, validate, transform, and store data** using Python — across common file formats and basic database workflows.

---

## Contents

- **Examples** — code shown during lectures (step-by-step)
- **Exercises** — tasks for independent practice
- **Mini-projects** — small end-to-end workflows (load → process → store)

---

## Topics Covered

- Working with files and paths
- Text & binary files
- CSV reading/writing (including headers, `DictReader/DictWriter`, newline handling)
- Excel basics (reading/writing)
- JSON and XML parsing/serialization
- Data cleaning & basic validation patterns
- Intro to databases (SQLite) and basic CRUD
- Practical storage patterns for small applications
